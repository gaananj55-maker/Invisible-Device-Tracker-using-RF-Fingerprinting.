# Invisible Device Tracker using RF Fingerprinting

## Objective
To identify and track wireless devices using RF fingerprinting and RSSI analysis.

## Features
- Device identification using ML
- RSSI-based tracking
- Passive detection (no device cooperation needed)

## Hardware Used
- RTL-SDR Dongle
- Antenna
- Laptop

## Software Used
- Python
- NumPy, SciPy
- Scikit-learn
